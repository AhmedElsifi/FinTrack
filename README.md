# 💱 Currency Converter

A modern currency converter built with React and Tailwind CSS that provides real-time currency conversion, live market data, historical comparisons, favorites, and conversion history.

## ✨ Features

- 🔄 Real-time currency conversion
- 🌍 Support for 150+ currencies
- 📈 Live market ticker with exchange rate movements
- 📊 Weekly market comparison percentages
- 🔁 One-click currency swap
- ⭐ Favorite currency pairs
- 📝 Conversion history log
- 📱 Responsive design
- ⚡ Fast and lightweight interface
- 💾 Local storage persistence

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/AhmedElsifi/FinTrack.git
```

### 2. Navigate to the project

```bash
cd fintrack
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

### 5. Open in browser

```txt
http://localhost:5173
```

## 🔥 Main Features Breakdown

### Currency Conversion

Convert between currencies using up-to-date exchange rates.

### Currency Swapping

Instantly swap the source and target currencies while preserving conversion values.

### Live Markets

Displays market movements for multiple currencies and compares current rates against historical data.

### Favorites

Save frequently used currency pairs for quick access.

### History Log

Track previous conversions and revisit them later.

## 💾 Local Storage

The application stores user preferences, favorites, and conversion history locally to provide a seamless experience across sessions.
